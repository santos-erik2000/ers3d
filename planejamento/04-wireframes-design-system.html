02 Tipografia

Diferente destes documentos de planejamento (que usam serifa no título por tratamento editorial), a interface do produto usa apenas sans-serif — é uma ferramenta operacional, escaneada rapidamente, não lida como texto corrido. Pilha de sistema por padrão (sem dependência de fonte externa); um logotipo/fonte de marca pode ser adicionado depois sem afetar esta estrutura.

05 Componentes principais

Sidebar

Ícone + rótulo, item ativo com fundo azul-claro e barra lateral de 3px; colapsável em tablet.

Header

Breadcrumb à esquerda, busca central, notificações + avatar à direita.

KPI card

Rótulo, valor grande (tabular-nums), variação vs. período anterior, ícone semântico discreto.

Tabela de dados

Cabeçalho fixo ao rolar, ação em hover na linha, paginação simples, ordenação por coluna.

Kanban card

Cliente/empresa, valor, avatar do responsável, prazo com status-pill, tags, indicador de dias na etapa.

Formulário

Rótulo acima do campo, texto de ajuda abaixo, erro em vermelho com ícone — nunca só a borda muda de cor.

Currency input

Prefixo R$ fixo, sempre 2 casas decimais, alinhado à direita, tabular-nums.

Percentage input

Sufixo %, aceita "20" e converte para 0,20 internamente — nunca pede o decimal do usuário.

File uploader

Área de arrastar-e-soltar + lista com ícone por tipo de arquivo (STL/STEP/PDF/imagem) e progresso.

Modal

Largura máx. 560px, título + corpo + ações no rodapé (ação primária à direita).

Drawer

Painel lateral para ver detalhe rápido sem sair da lista/quadro atual.

Toast

Canto superior direito, some sozinho em ~4s, cor semântica conforme o resultado.

Timeline

Histórico vertical com ponto + linha — usado na página 360° do cliente e no card do Kanban.

Empty state

Ícone leve, uma frase objetiva, um call-to-action — nunca uma tela em branco sem explicação.

Skeleton

Placeholder cinza pulsante no formato real do conteúdo, nunca um spinner genérico central.

Confirm dialog

Exige clique consciente para ações sensíveis (excluir, reprovar, fechar ciclo) — nunca some sozinho.

Command palette

Atalho de teclado para navegação/ação rápida — item de conveniência, não bloqueia nada se não for usado.
